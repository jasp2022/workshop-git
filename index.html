Control de versiones

Un control de versiones es un sistema que registra cada cambio que se realiza en el código fuente de un proyecto. Esto te permite tener un histórico de todos los cambios producidos en sus ficheros, saber quién lo hizo y cuándo.

¿Que es GIT?

Git es un sistema de control de versiones distribuido, gratuito y de código abierto. Fue diseñado por Linus Torvalds, el creador del kernel de Linux, en 2005. Git es uno de los sistemas de control de versiones más populares y ampliamente utilizados en el mundo.
